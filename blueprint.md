# Blueprint

## Purpose and capabilities

Latravel Tours is an Astro-based site for browsing curated Thailand travel experiences with fast, static-first rendering.

## Current project outline

- Astro 5 site with Tailwind CSS
- Main entry page at `src/pages/index.astro`
- Firebase Studio / IDX preview support configured in `.idx/dev.nix`
- Production validation currently uses `npm run build`
